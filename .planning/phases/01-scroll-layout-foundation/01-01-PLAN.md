# Plan 01-01: Initialize Project Configuration and Smooth Scrolling Sync

## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Memory leaks on route change / tick re-binding | Medium | Revert GSAP contexts and clean up Lenis ticker bindings on component unmount. |
| Server-side rendering (SSR) failures | High | Dynamic import Lenis wrapper or check `typeof window !== 'undefined'` before execution. |

### Task 2: Implement Lenis Smooth Scroll Provider and Ticker Sync

Configure Lenis inertial smooth scroll and synchronize it to the GSAP ticker loop.

<read_first>
- `app/layout.tsx`
- `lib/gsap.ts` [NEW]
- `components/ui/SmoothScrollProvider.tsx` [NEW]
</read_first>

<acceptance_criteria>
- Direct window scroll input triggers smooth inertial scrolling.
- GSAP tickers are registered correctly and synced to Lenis.
- No SSR errors occur on load.
</acceptance_criteria>

<action>
1. Create `lib/gsap.ts`:
   - Import `gsap` and register `ScrollTrigger` from `gsap/all`.
   - Export custom GSAP/ScrollTrigger configuration.
2. Create client component `components/ui/SmoothScrollProvider.tsx`:
   - Import `Lenis` from `@studio-freight/lenis`.
   - Instantiate Lenis.
   - Sync scroll ticker:
     ```typescript
     lenis.on('scroll', ScrollTrigger.update);
     gsap.ticker.add((time) => {
       lenis.raf(time * 1000);
     });
     gsap.ticker.lagSmoothing(0);
     ```
   - Wrap children in Lenis react context provider.
3. Update `app/layout.tsx` to wrap the `body` children inside `<SmoothScrollProvider>`.
</action>

### Task 3: Setup Accessibility Fallbacks

Implement support for reduced motion settings.

<read_first>
- `components/ui/SmoothScrollProvider.tsx`
</read_first>

<acceptance_criteria>
- Users with OS `prefers-reduced-motion: reduce` have Lenis smooth scroll disabled automatically.
</acceptance_criteria>

<action>
1. Inside `components/ui/SmoothScrollProvider.tsx`, check media query `(prefers-reduced-motion: reduce)`.
2. If matched, initialize Lenis with options `{ duration: 0 }` or do not instantiate Lenis to preserve native instant scroll.
</action>
