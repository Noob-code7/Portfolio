# Plan 01-03: Circular Clip-Path Theme Toggle and Noise Overlay

## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Clip-path render flashes / jumps | Medium | Ensure state updates and clip-path CSS transitions occur inside a unified animation frame. |
| Noise overlay blocks click inputs | High | Set `pointer-events: none` on the noise overlay wrapper container. |

## Tasks

### Task 1: Build Theme Toggle and Circular Clip-Path Wipe

Create the theme switch state and click transition.

<read_first>
- `components/ui/ThemeToggle.tsx` [NEW]
- `app/globals.css`
</read_first>

<acceptance_criteria>
- Theme toggles light/dark values successfully in CSS classes.
- Clicking the toggle button triggers a circular clip-path transition wiping outward from button center.
</acceptance_criteria>

<action>
1. Create `components/ui/ThemeToggle.tsx`:
   - Initialize theme state (dark/light) reading system preferences or localstorage.
   - Use custom transition animation on toggle:
     - Capture click coordinate x and y relative to viewport.
     - Calculate radius to furthest corner.
     - Apply transient overlay with `clip-path` animating from `circle(0px at x y)` to `circle(maxRadius at x y)`.
     - Toggle `dark` class on root HTML element.
</action>

### Task 2: Configure Global Noise Overlay

Add the low-opacity styling grain layer.

<read_first>
- `app/layout.tsx`
- `app/globals.css`
</read_first>

<acceptance_criteria>
- Subtle grain pattern is visible over the page.
- Mouse clicks pass through the noise layer to buttons underneath.
</acceptance_criteria>

<action>
1. Define a tiling noise style in `app/globals.css` using a low-opacity base64 encoded noise PNG or transparent styling background:
   ```css
   .noise-overlay {
     position: fixed;
     top: 0;
     left: 0;
     width: 100vw;
     height: 100vh;
     background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noiseFilter'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.65' numOctaves='3' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noiseFilter)'/%3E%3C/svg%3E");
     opacity: 0.02;
     pointer-events: none;
     z-index: 9999;
   }
   ```
   *(Note: Base64/Inline SVG noise represents a reliable, high-performance tiling image format).*
2. Add `.noise-overlay` div inside `app/layout.tsx`.
</action>
